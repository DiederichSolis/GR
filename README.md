RUNNING DE PROJECT
<p>
1.) clonar repositorio
</p>

```
gti clone https://github.com/DiederichSolis/GR.git
```

<p>
2.) Pasarse a la carpeta GR
</p>

```
cd GR
```

<p>
3.) Realizar el build
</p>

```
cargo build
```

4.) Correrlo
</p>

```
cargo run 
```
